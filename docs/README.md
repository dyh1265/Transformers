# Documentation

## Tutorials (notebooks)

Run from the repo root or from `docs/` (notebooks add `src/` to `sys.path`).

| Notebook | Topic |
|----------|--------|
| [tokenizer_tutorial.ipynb](tokenizer_tutorial.ipynb) | HF byte-level BPE only (`HFByteBPETokenizer`), state roundtrip, training CLI |
| [imdb_sentiment_tutorial.ipynb](imdb_sentiment_tutorial.ipynb) | IMDB formatting, `load_imdb_sentiment`, tokenizers, `create_dataloaders` |
| [sampling_tutorial.ipynb](sampling_tutorial.ipynb) | Greedy / top-k / top-p, temperature, repetition penalty, `generate()` |
| [vanilla_vs_inter_block_decoder.ipynb](vanilla_vs_inter_block_decoder.ipynb) | Vanilla vs inter-block decoder stack (with [companion markdown](vanilla_vs_inter_block_decoder.md)) |
